ttylog

ttylog - a serial port logger
Navigation: Home | Development

The program ttylog is a serial port logger which can be used to print everything to stdout that comes from a serial device. The device and the baud rate can be specified on the command line. It was originally written and packaged for Debian by Tibor Koleszar and is now maintained in Debian and Ubuntu by Robert James Clay.
