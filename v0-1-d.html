0.1.d

Version 0.1.d will be used for the next updated version of ''ttylog'' itself. Planned and/or possible changes are currently as follows:
- Changes to the build system
- Move ttylog to /usr/bin from /usr/sbin?
- Provide an option to time-stamp the output?
- Update ttylog man page
- Add option to not output a newline character?
- Add option to output only a single character at a time?

More information about v0.1.d development may be found on the ttylog Wiki page.
